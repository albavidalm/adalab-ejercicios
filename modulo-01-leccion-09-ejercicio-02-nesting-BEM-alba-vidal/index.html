Utilizando el operador "&" hay que reescribir el siguiente código con Sass. El reto está en escribir la palabra card una sola vez.

De CSS a SCSS
CSS SCSS

.card {
  width: 100px;
  height: 100px;
  margin: 10px;
}
.card:hover {
  box-shadow: 10px 10px 5px 0px rgba(0,0,0,0.75);
}
.card--dark {
  background-color: #000;
  color: #fff;
}
.card--light {
  background-color: #fff;
  color: #000;
}

.card{
  width: 100px;
  height: 100px;
  margin: 10px;
  &:hover {
      box-shadow: 10px 10px 5px 0px rgba(0,0,0,0.75);
  }
  &--dark {
      background-color: #000;
      color: #fff;
  }
  &--light {
      background-color: #fff;
      color: #000;
  }
}
